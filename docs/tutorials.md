# OVC Tutorials

Step-by-step tutorials to master the Overlap Violation Checker.

**Version:** v3.0.0

> [!TIP]
> **Tutorials vs Examples:** Tutorials provide **guided, narrative learning paths** that build your understanding progressively. For **quick, copy-paste code snippets** and ready-to-use recipes, see [Examples](examples.md).

## Learning Path Recommendation

1. **New Users:** Start with Tutorials 1 & 2 to understand the basics.
2. **GIS Analysts:** Focus on Tutorials 3, 5, and 6 for day-to-day operations.
3. **Data Engineers:** Jump to Tutorials 4 and 7 to build automated pipelines.
4. **Road Network Analysis:** Follow Tutorial 8 for Road QC workflows.
